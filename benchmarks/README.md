Code to reproduce the benchmarks in the paper.

For each folder, first install the dependencies with `pip install -r requirements.txt` or `pip install -e .`, then follow the instructions in the README or run the scripts `run.sh`.

To reproduce the results of SR, please refer to the original implementation by the authors https://github.com/RUCKBReasoning/SubgraphRetrievalKBQA (all rights reserved, not open source).
